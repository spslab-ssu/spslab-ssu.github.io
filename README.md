# SPS Lab @ SSU — Lab Website

숭실대학교 스마트 생산시스템(SPS) 연구실 홈페이지 저장소입니다.
Jekyll + GitHub Pages 기반이며, 실제 사이트는 <https://spslab-ssu.github.io> 에서 볼 수 있습니다.
